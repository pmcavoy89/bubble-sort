Bubble Sort

Bubble Sort is an algorithm that allows a list of elements to be put into sorted order. This is done by swapping the element that are out of order.

Given a size of n elements, the algorithm runs O(n)^2 time. Let's take a look at what that means.

Inputs

Example List Input: [0, 9, 8, 7, 6, 5, 4, 3, 2, 1]

Key Points:

  • Unordered List
  • Number of elements to be small
  • Not efficient
  • Quick and easy (but there's other approaches)
